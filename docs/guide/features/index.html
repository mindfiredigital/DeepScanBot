Features

DeepScanBot comes with a comprehensive set of features for web crawling and scraping.

Core Features​

Concurrent Crawling​

Multi-threaded architecture with configurable concurrency limits for optimal performance.

# Set concurrency to 10
go run ./apps/cli -url https://example.com -concurrency 10

Configurable Crawl Depth​

Control how deep the crawler explores linked pages.

# Single page only
go run ./apps/cli -url https://example.com -depth 0

# Full site (3 levels deep)
go run ./apps/cli -url https://example.com -depth 3

Robots.txt Compliance​

Respects robots.txt rules with an option to ignore them.

# Ignore robots.txt
go run ./apps/cli -url https://example.com -ignore-robots

Retry Logic​

Built-in retry mechanism with exponential backoff for transient failures.

# Retry up to 3 times with 2 second backoff
go run ./apps/cli -url https://example.com -retries 3 -retry-backoff 2s

Advanced Features​

Content Filtering​

Filter crawled pages by MIME types and size limits. Note: the value must be quoted as a single argument.

# Only HTML pages
go run ./apps/cli -url https://example.com -content-types "text/html"

# Limit page size to 1MB
go run ./apps/cli -url https://example.com -size 1024

Proxy Support​

Route traffic through HTTP/HTTPS proxies.

go run ./apps/cli -url https://example.com -proxy http://127.0.0.1:8080

Sitemap Discovery​

Automatically discover and crawl URLs from sitemap.xml.

go run ./apps/cli -url https://example.com -sitemap

Resume Mode​

Resume interrupted crawls without recrawling already visited URLs.

# First crawl
go run ./apps/cli -url https://example.com -output my-crawl

# Resume
go run ./apps/cli -url https://example.com -resume -output my-crawl

Cross-Domain Crawling​

Optionally follow links to external domains.

go run ./apps/cli -url https://example.com -cross-domain

Multiple Output Formats​

Export results in JSON or plain text format.

# Text output (default)
go run ./apps/cli -url https://example.com -output results

# JSON output
go run ./apps/cli -url https://example.com -json -output results

Politely Crawl​

Configure delay between requests to avoid overwhelming servers.

# Wait 1 second between requests
go run ./apps/cli -url https://example.com -delay 1s
