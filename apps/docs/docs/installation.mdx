# Installation

Get DeepScanBot up and running in minutes.

---

## Prerequisites

Before you begin, ensure you have the following installed:

- **Go** (version 1.22 or higher) — [Download Go](https://go.dev/dl/)
- **Git** — [Download Git](https://git-scm.com/downloads)

## Install from Source

```bash
# Clone the repository
git clone https://github.com/mindfiredigital/DeepScanBot.git

# Navigate to the project
cd DeepScanBot

# Download dependencies
go mod download

# Build the binary
go build -o deepscanbot ./apps/cli
```

The build process produces a single, self-contained binary named `deepscanbot` in your current directory. No runtime dependencies required.

## Run Without Building

You can also run DeepScanBot directly using `go run` — no build step needed:

```bash
go run ./apps/cli -url https://example.com -depth 2 -json
```

## Verify Installation

```bash
# Show help and available flags
./deepscanbot -h

# Run the test suite
go test ./...

# Quick smoke test
./deepscanbot -url https://example.com -depth 0 -json
```

## Install Development Tools (Optional)

If you plan to contribute to the project:

```bash
# Code formatting
go install mvdan.cc/gofumpt@latest

# Import management
go install github.com/daixiang0/gci@latest

# Linting
go install github.com/golangci/golangci-lint/cmd/golangci-lint@latest

# Git hooks
go install github.com/evilmartians/lefthook@latest && lefthook install
```
