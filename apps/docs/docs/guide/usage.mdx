# Usage Guide

Learn how to use DeepScanBot effectively for your web crawling needs.

## Basic Usage

The simplest way to crawl a website:

```bash
go run ./apps/cli -url https://example.com
```

This will:
- Crawl the start page
- Follow links up to 2 levels deep
- Output results to `crawler_results.txt`

## Common Use Cases

### Single Page Crawl

Crawl only the specified URL without following links:

```bash
go run ./apps/cli -url https://example.com -depth 0
```

### Full Site Crawl

Crawl an entire site with greater depth:

```bash
go run ./apps/cli -url https://example.com -depth 5 -concurrency 20
```

### JSON Output

Export results in JSON format for programmatic processing:

```bash
go run ./apps/cli -url https://example.com -json -output results
```

### Crawl with Proxy

Route traffic through a proxy server:

```bash
go run ./apps/cli -url https://example.com -proxy http://127.0.0.1:8080
```

### Content-Type Filtering

Only download specific file types:

```bash
# Only HTML pages
go run ./apps/cli -url https://example.com -content-types "text/html"

# HTML and PDF documents
go run ./apps/cli -url https://example.com -content-types "text/html,application/pdf"

# All content types
go run ./apps/cli -url https://example.com -content-types "*/*"
```

### Unique URLs Only

Avoid processing duplicate URLs:

```bash
go run ./apps/cli -url https://example.com -u
```

### Resume a Crawl

Stop and resume a crawl without recrawling:

```bash
# Initial crawl
go run ./apps/cli -url https://example.com -depth 3 -output my-crawl

# Resume (skips already crawled URLs)
go run ./apps/cli -url https://example.com -depth 3 -resume -output my-crawl
```

### Politely Crawl

Add delays to avoid overwhelming servers:

```bash
go run ./apps/cli -url https://example.com -delay 500ms -host-concurrency 2
```

## Output Formats

### Text Output (Default)

```
https://example.com [status=200] [result=passed]
https://example.com/about [status=200] [result=passed]
https://example.com/contact [result=discovered]
https://example.com/admin [result=skipped] [skipped=disallowed by robots.txt]
```

### JSON Output

```json
{
  "start_url": "https://example.com",
  "started_at": "2024-01-01T00:00:00Z",
  "finished_at": "2024-01-01T00:00:05Z",
  "summary": {
    "total": 10,
    "passed": 8,
    "failed": 1,
    "skipped": 1,
    "errors": 1
  },
  "urls": [
    {
      "url": "https://example.com",
      "source": "href",
      "depth": 0,
      "status_code": 200,
      "content_type": "text/html",
      "result": "passed",
      "attempts": 1
    }
  ],
  "skipped": [
    {
      "url": "https://example.com/admin",
      "result": "skipped",
      "skipped_reason": "disallowed by robots.txt"
    }
  ]
}
```

## Exit Codes

| Code | Meaning |
|------|---------|
| `0` | Success |
| `1` | Error (invalid arguments, crawl failure) |

## Examples

### Crawl Documentation Site

```bash
go run ./apps/cli -url https://docs.example.com -depth 3 -sitemap -json -output docs-crawl
```

### Monitor a Website

```bash
go run ./apps/cli -url https://example.com -depth 1 -u -output monitor -dr
```

### Scrape PDF Documents

```bash
go run ./apps/cli -url https://example.com -depth 2 -content-types "application/pdf" -size 10240 -output pdfs
