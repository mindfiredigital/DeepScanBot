# Introduction

DeepScanBot is a powerful, feature-rich web crawler built with Go, designed for efficient and configurable web scraping.

## What is DeepScanBot?

A high-performance web crawler that combines robust crawling capabilities with flexible configuration options. Built with Go, it delivers exceptional performance and reliability for web scraping tasks of any scale.

## Key Features

- **Concurrent Crawling**: Multi-threaded architecture with configurable concurrency limits
- **Robots.txt Compliance**: Respects robots.txt rules with option to ignore
- **Configurable Depth**: Control crawl depth from single page to full site exploration
- **Retry Logic**: Built-in retry mechanism with exponential backoff
- **Content Filtering**: Filter by MIME types and page size
- **Proxy Support**: HTTP/HTTPS proxy support
- **Sitemap Discovery**: Automatic sitemap.xml parsing and crawling
- **Resume Mode**: Ability to resume interrupted crawls
- **Multiple Output Formats**: JSON and text output formats
- **Cross-Domain Crawling**: Optional cross-domain link following

## Why Choose DeepScanBot?

Modern web crawling requires more than just fetching pages. DeepScanBot provides the right balance of performance, configurability, and reliability to handle any web scraping task efficiently.
