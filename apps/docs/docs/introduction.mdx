# Welcome to DeepScanBot

A powerful, production-ready web crawler built with Go — designed for performance, reliability, and developer happiness.

---

## What is DeepScanBot?

DeepScanBot is a **feature-rich, concurrent web crawler** that recursively crawls websites, respects `robots.txt`, handles rate-limiting, and produces comprehensive JSON or text reports. Built entirely in Go, it delivers exceptional performance as a single, self-contained binary.

Whether you're auditing a website, collecting data for analysis, or monitoring changes across thousands of pages, DeepScanBot provides the tools you need — out of the box.

## Why DeepScanBot?

| Feature | DeepScanBot | Other Crawlers |
|---------|-------------|----------------|
| **Language** | Go (single binary) | Often Python/Node (requires runtime) |
| **Concurrency** | Built-in, configurable | Often limited or complex setup |
| **Rate Limiting** | Per-host + global semaphores | Manual or absent |
| **Robots.txt** | Automatic + bypass option | Varies |
| **Retry Logic** | Exponential backoff + Retry-After | Often basic or missing |
| **Output** | JSON / Text with full analytics | Usually basic |
| **Resume Mode** | Built-in | Rare |
| **Sitemap** | Auto-discovery + nested | Rare |

## Quick Start

```bash
# Run without building (requires Go)
go run ./apps/cli -url https://example.com -depth 2 -json

# Or build once and use anywhere
go build -o deepscanbot ./apps/cli
./deepscanbot -url https://example.com -depth 2 -json
```
