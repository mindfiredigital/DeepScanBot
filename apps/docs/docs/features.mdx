# Features

DeepScanBot comes with a comprehensive set of features for web crawling and scraping.

## Core Features

### Concurrent Crawling
Multi-threaded architecture with configurable concurrency limits for optimal performance.

```bash
# Set concurrency to 10
./deepscanbot -url https://example.com -concurrency 10
```

### Configurable Crawl Depth
Control how deep the crawler explores linked pages.

```bash
# Single page only
./deepscanbot -url https://example.com -depth 0

# Full site (3 levels deep)
./deepscanbot -url https://example.com -depth 3
```

### Robots.txt Compliance
Respects robots.txt rules with an option to ignore them.

```bash
# Ignore robots.txt
./deepscanbot -url https://example.com -ignore-robots
```

### Retry Logic
Built-in retry mechanism with exponential backoff for transient failures.

```bash
# Retry up to 3 times with 2 second backoff
./deepscanbot -url https://example.com -retries 3 -retry-backoff 2s
```

## Advanced Features

### Content Filtering
Filter crawled pages by MIME types and size limits.

```bash
# Only HTML and PDF files
./deepscanbot -url https://example.com -content-types "text/html,application/pdf"

# Limit page size to 1MB
./deepscanbot -url https://example.com -size 1024
```

### Proxy Support
Route traffic through HTTP/HTTPS proxies.

```bash
./deepscanbot -url https://example.com -proxy http://127.0.0.1:8080
```

### Sitemap Discovery
Automatically discover and crawl URLs from sitemap.xml.

```bash
./deepscanbot -url https://example.com -sitemap
```

### Resume Mode
Resume interrupted crawls without recrawling already visited URLs.

```bash
# First crawl
./deepscanbot -url https://example.com -output my-crawl

# Resume
./deepscanbot -url https://example.com -resume -output my-crawl
```

### Cross-Domain Crawling
Optionally follow links to external domains.

```bash
./deepscanbot -url https://example.com -cross-domain
```

### Multiple Output Formats
Export results in JSON or plain text format.

```bash
# Text output (default)
./deepscanbot -url https://example.com -output results

# JSON output
./deepscanbot -url https://example.com -json -output results
```

### Politely Crawl
Configure delay between requests to avoid overwhelming servers.

```bash
# Wait 1 second between requests
./deepscanbot -url https://example.com -delay 1s
```

## All CLI Flags

| Flag | Default | Description |
|------|---------|-------------|
| `-url` | `""` | Starting URL |
| `-depth` | `2` | Maximum crawl depth |
| `-timeout` | `2` | Request timeout (seconds) |
| `-proxy` | `""` | Proxy URL |
| `-json` | `false` | JSON output |
| `-size` | `-1` | Page size limit (KB) |
| `-dr` | `false` | Disable redirects |
| `-s` | `false` | Show URL source |
| `-insecure` | `false` | Disable TLS verification |
| `-u` | `false` | Unique URLs only |
| `-concurrency` | `0` | Max concurrent requests |
| `-host-concurrency` | `0` | Max concurrent requests per host |
| `-content-types` | `"text/html"` | Allowed MIME types |
| `-output` | `"crawler_results"` | Output filename |
| `-ignore-robots` | `false` | Ignore robots.txt |
| `-cross-domain` | `false` | Follow external links |
| `-retries` | `0` | Retry attempts |
| `-retry-backoff` | `"1s"` | Retry backoff duration |
| `-delay` | `"0s"` | Politeness delay |
| `-sitemap` | `false` | Discover sitemap.xml |
| `-resume` | `false` | Resume mode |
| `-h` | `false` | Show help |
